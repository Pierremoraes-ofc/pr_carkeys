# 🔑 pr_carkeys — Vehicle Key System

> A complete vehicle key system for FiveM. Lock, unlock, hotwire, carjack, key bags and much more.

---

## ✨ Features

- 🔐 **4 key types** — original, copy, temporary and single-use
- 👜 **Key bags** — store and organize your keys
- 🚗 **Hotwire** — start vehicles without a key via minigame
- 🔫 **Carjack** — rob NPC vehicles with weapon at gunpoint
- 🔊 **3D sounds** — lock/unlock sounds heard by nearby players
- 👮 **Police system** — officers can access any vehicle
- ⏱️ **Auto expiration** — timed keys removed automatically
- 🌐 **Multi-framework** — QBCore, QBX, ESX, ox_core, ND_Core
- 📦 **Multi-inventory** — ox_inventory and qb-inventory
- 🗄️ **Multi-database** — oxmysql, ghmattimysql, mysql-async

## ⚡ Quick Start

```
1. Add pr_carkeys to your resources folder
2. Add `ensure pr_carkeys` to server.cfg
3. Add items to your inventory (ox_inventory or qb-inventory)
4. Restart server — the database table is created automatically
```
